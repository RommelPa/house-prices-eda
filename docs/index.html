House Prices — Exploratory Data Analysis

This project explores the Ames Housing Dataset to understand the main factors that influence residential property prices. The analysis focuses on data quality assessment, feature exploration, statistical relationships, and outlier detection before building predictive models.

Dataset

Dataset: Ames Housing Dataset
Source: Kaggle House Prices Competition

The dataset contains information about residential home sales in Ames, Iowa with 81 features describing house characteristics.

Project Workflow

  1. Data Overview
  2. Data Quality Audit
  3. Feature Distribution Analysis
  4. Correlation Analysis
  5. Outlier Detection
  6. EDA Summary and Insights

Key Insights

  • OverallQual (overall house quality) is the strongest predictor of house prices.
  • GrLivArea (above ground living area) shows strong positive correlation with price.
  • Several variables related to house size strongly influence property value.
  • Neighborhood plays an important role in price variation across properties.
  • Extreme outliers were detected and removed to improve analysis robustness.
