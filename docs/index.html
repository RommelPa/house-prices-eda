House Prices — Exploratory Data Analysis

This project explores the Ames Housing dataset to identify the main factors influencing residential property prices. The analysis focuses on data quality assessment, feature exploration, statistical relationships, and outlier detection.

Dataset

Dataset: Ames Housing Dataset
Source: Kaggle House Prices Competition
Observations: 1460
Features: 81

Key Insights

  • House quality (OverallQual) is the strongest predictor of house prices
  • Living area (GrLivArea) strongly correlates with price
  • Several size-related variables influence house value
  • Neighborhood significantly affects property prices
  • Outliers were detected and removed
